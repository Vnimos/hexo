hexo命令

  • init 新建一个网站
    若未指定folder则默认在当前目录下创建

  • new 新建一篇文章
    若未指定layout则默认 _config.yml中的 default_layout 指定的参数

  • generate 生成静态文件
    -d,–deploy　　　　　文件生成后立即部署网站
    -w,–watch　　　　　监视文件变动

  • publish 发表草稿

  • server 启动服务器
    -i　　　　　　　　　　指定IP地址
    -p,–port　　　　　　　指定端口号（默认为4000）
    -s,–static　　　　　　　只使用静态文件
    -l,–log　　　　　　　　启动日志记录，使用覆盖记录格式

  • deploy 部署网站
    -g,–generate　　　　　　　部署之前预先生成的静态文件

  • clean 清除缓存文件（db.json）和已生成的静态文件(public)

  • list 列出网站资料
